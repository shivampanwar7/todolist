# Task Manager App

A simple React.js Task Manager app where users can add, delete, and mark tasks as completed.

## Features
✅ Add tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Routing using React Router  
✅ Deployed on Vercel  

## Installation
```sh
git clone https://github.com/yourusername/task-manager.git
cd task-manager
npm install
npm run dev
